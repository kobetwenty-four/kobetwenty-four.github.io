搭建项目

1.命令

可以通过附加的命令行选项直接指定项目名称和你想要使用的模板

vite

npm 6.x

npm create vite@latest my-vue-app –template vue

npm 7+, extra double-dash is needed:

npm create vite@latest my-vue-app – –template vue

yarn

yarn create vite my-vue-app –template vue

pnpm

pnpm create vite my-vue-app –template vue

2.启动

cd 到项目名目录文件

npm install

npm run dev

安装 threejs

npm install three -S

安装 vue-router #### vue-router

npm install vue-router@4

src 下面新建文件夹 views

views 下面建一个 Home.vue 文件
views 下面建一个 router 文件夹
router 下面建一个 index.js

配置 vite.config.js

配置路由

3.我的完整示例

npm -v 8.11.0

router –> index.js

main.js

vite.config.js
